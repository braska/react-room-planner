# Room planner

![preview](preview.jpg)

## Technologies

* React
* Redux
* Immutable
* Babel
* ES2015
* JavaScript experimental features (stage-0)
* Webpack
* SASS

## Requirements

* NodeJS >= 5.4.1
* NPM

## Run

Inside project folder:

```
npm install
npm run serve
```

## Run in dev mode

```
npm run serve-dev
```

## Only build sources

```
npm run build
```
